## Refonte du thème Joomla responsive

Service Innovation Group (www.sigeurope.fr) souhaitait moderniser son site web avec une refonte complète de son thème Joomla. Notre mission principale était de rendre le site entièrement responsive pour offrir une expérience utilisateur optimale sur tous les appareils, des ordinateurs de bureau aux smartphones.

## Technologies utilisées

Le projet a été réalisé en utilisant HTML5 et CSS3 pour l'intégration responsive des maquettes Photoshop. Nous avons également implémenté des animations JavaScript pour les menus, améliorant ainsi l'expérience utilisateur et la navigation sur le site. L'ensemble a été intégré dans un environnement Joomla.

## Résultats obtenus

La refonte du thème a permis à Service Innovation Group de disposer d'un site web moderne, accessible sur tous les appareils grâce à l'approche responsive. Les animations des menus ont contribué à une navigation plus intuitive et agréable pour les utilisateurs. Le site est désormais parfaitement adapté aux consultations sur mobile et tablette, élargissant ainsi sa portée auprès des utilisateurs.
